# Bug Fixes: Weapons

Fix weapon-side bugs and weapon-specific boon interactions that can affect speedrun attempts.

Part of the [Speedrun modpack](https://thunderstore.io/c/hades-ii/p/adamant/Speedrun_Core/).

## What It Does

Bug Fixes: Weapons collects fixes for weapon behavior and weapon-specific boon interactions that can behave incorrectly in live runs.

The module currently includes:

- Extra Dose Fix
  Fixes Extra Dose interactions with Coat second punch and dash strike.
- Aspect of Selene Fix
  Registers Skyfall as a Hex when starting a new run, allowing Path of Stars to be offered directly without needing to pick a Selene's Gift first. Skyfall also always starts at full Moonglow.
- Axe and Blade Omega Channel Fix
  Fixes Axe omega attack and Blade omega special not benefiting correctly from channeling bonus.
- Tidal Ring Fix
  Fixes Tidal Ring failing to hit the same mob twice with Circe Staff.

Use it when you want weapon-side bug fixes applied without changing broader routing or utility behavior in the pack.

## Installation

Install via r2modman.

This module is usually installed as part of the full [Speedrun modpack](https://thunderstore.io/c/hades-ii/p/adamant/Speedrun_Core/), where it appears in the shared Speedrun UI with the other speedrun-focused modules.
